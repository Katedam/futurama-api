Futurama API

Documentation

  • Get All Quotes > futuramaapi.herokuapp.com/api/quotes
  • Get 'count' Quotes > futuramaapi.herokuapp.com/api/quotes/<count>
  • Get All Quotes By A Character > futuramaapi.herokuapp.com/api/characters/<character name>
  • Get 'count' Quotes By A Character > futuramaapi.herokuapp.com/api/characters/<character name>/<count>
